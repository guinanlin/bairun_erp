# 半成品列表 — 后端支持说明

**文档版本**：2026-03  
**关联页面**：`/scm/mes/semi-finished`（半成品列表）  
**关联接口**：`bairun_erp.utils.api.stock.blank_list`（已支持毛坯 + 半成品）

---

## 1. 是否支持半成品？

**结论：已支持。** 通过参数 `list_type=semi_finished` 使用同一套接口。

| 维度     | 毛坯（`list_type=blank` 或默认） | 半成品（`list_type=semi_finished`） |
|----------|----------------------------------|-------------------------------------|
| 到库数   | 入 **毛坯仓**（`毛坯 - B`）的 SE，通过 PR 取销售订单 | 入 **半成品仓**（`半成品 - B`）的 SE（即调拨 毛坯→半成品），project_no 从 SE.custom_customer_order |
| 委外数   | Material Transfer **毛坯仓 → 半成品仓** | Material Transfer **半成品仓 → 成品仓**（`半成品 - B` → `成品 - B`） |
| 待委外   | 到库数 > 0 且 委外余数 > 0 | 同上（半成品仓视角） |
| 已委外   | 委外数 > 0 | 同上（半成品→成品 调拨数 > 0） |

---

## 2. 前端对接方式

- **接口**：与毛坯相同  
  - 待委外：`POST ... get_pending_outsourcing_list`  
  - 已委外：`POST ... get_outsourced_list`
- **请求**：在 `json_data` 中增加 **`list_type`**：
  - 毛坯页：不传或传 `"list_type": "blank"`
  - 半成品页：传 `"list_type": "semi_finished"`
- **响应**：与毛坯一致，沿用《毛坯列表-前端调用说明》中的 `message`、`total_count` 及行字段；半成品时 `warehouse` 为 `半成品 - B`。

---

## 3. 参数约定

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `list_type` | string | 否 | `blank`（默认）毛坯；`semi_finished` 半成品。 |
| 其它 | — | — | 与毛坯一致：`limit_start`、`limit_page_length`、`order_by`、`search_project_no`、`search_outsourcing_supplier` 等。 |

---

## 4. 仓库约定（与站点一致）

- 毛坯仓：`毛坯 - B`
- 半成品仓：`半成品 - B`
- 成品仓：`成品 - B`（半成品委外目标仓）
