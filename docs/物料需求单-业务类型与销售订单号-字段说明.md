# 物料需求单（Material Request）自定义字段说明

以下字段在 `bairun_erp/custom/material_request.json` 中维护，站点需执行 `bench migrate` 后与数据库同步。

## 1. 业务类型（`custom_business_type`）

- **类型**：下拉（Select）
- **用途**：区分本张物料需求单对应的业务场景。
- **可选值**：
  - 毛坯转半成品
  - 半成品转成品
- **位置**：表头区域，在「物料需求类型」之后。

## 2. 销售订单号（`custom_sales_order_no`）

- **类型**：链接（Link），关联 **Sales Order**（销售订单）
- **用途**：标明该物料需求与哪一张销售订单相关；保存的是销售订单单据名（如 `SO-xxxxx`）。
- **位置**：表头区域，在「业务类型」之后。

---

若表单上看不到「业务类型」，请确认该自定义字段在系统中 **未勾选隐藏（Hidden）**，并已重新迁移或刷新自定义。
