# 原材料页面 - 接口字段说明

本文档描述 MES 原材料页面涉及的主要数据表及字段，供后端工程师提供 API 接口参考。

---

## 一、页面概览

- **页面路径**：`/scm/mes/raw-materials`
- **业务场景**：采购/入库/出库，按状态分 Tab：待入库、已入库、已出库
- **筛选条件**：采购单号（可选）

---

## 二、原材料明细主表

主列表展示的每条原材料记录。

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `id` | string | 是 | 行记录唯一标识 |
| `date` | string | 是 | 单据日期，格式 `yyyy-MM-dd` |
| `projectNo` | string | 是 | 采购单号 |
| `itemFullName` | string | 是 | 物料名称（存货全名） |
| `orderQty` | number | 是 | 采购订单数 |
| `unitPrice` | number | 是 | 采购单价 |
| `receivedQty` | number | 是 | 到库数 |
| `unreceivedQty` | number | 是 | 未到库数（= 采购订单数 - 到库数） |
| `inStockQty` | number | 是 | 在库数 |
| `supplierId` | string | 是 | 供应商 ID |
| `supplier` | string | 是 | 供应商名称 |
| `inventoryCost` | number | 是 | 库存成本 |
| `salesPrice` | number | 是 | 销售定价 |
| `warehouse` | string | 是 | 仓库 |
| `warehouseLocation` | string | 否 | 库位，可空 |
| `unit` | string | 是 | 单位（件/箱/个/套/kg/吨 等） |
| `workInstructionUrl` | string | 否 | 作业指导书图片 URL |
| `status` | string | 是 | 状态：`待入库` \| `已入库` \| `已出库` |

**说明**：

- `采购金额` 为计算字段：`orderQty * unitPrice`
- `unreceivedQty` 通常由 `orderQty - receivedQty` 计算得出

---

## 三、添加原材料物料弹窗

新增物料时弹窗表单字段。

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `itemName` | string | 是 | 物料名称 |
| `inventoryCost` | number | 否 | 库存成本，默认 0 |
| `salesPrice` | number | 否 | 销售定价，默认 0 |

---

## 四、参考数据（下拉选项）

### 4.1 采购单

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `id` | string | 采购单唯一标识 |
| `no` | string | 采购单号（展示用） |

### 4.2 物料（Item）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `id` | string | 物料唯一标识 |
| `fullName` | string | 物料名称（存货全名） |

### 4.3 供应商

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `id` | string | 供应商唯一标识 |
| `name` | string | 供应商名称 |

### 4.4 仓库

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `name` | string | 仓库名称 |

### 4.5 库位

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `name` | string | 库位名称 |

### 4.6 单位

可选值：`件`、`箱`、`个`、`套`、`kg`、`吨`（或其他 UOM）

---

## 五、建议接口清单

| 接口 | 方法 | 说明 |
|------|------|------|
| 获取原材料明细列表 | GET | 按状态、采购单号筛选，返回主表数据 |
| 新增/更新原材料明细 | POST/PUT | 单条或批量保存 |
| 统计入库 | POST | 将待入库改为已入库 |
| 出库 | POST | 将已入库改为已出库 |
| 获取采购单列表 | GET | 下拉选项 |
| 获取物料列表 | GET | 下拉选项 |
| 获取供应商列表 | GET | 下拉选项 |
| 获取仓库列表 | GET | 下拉选项 |
| 获取库位列表 | GET | 按仓库筛选，下拉选项 |

---

## 六、状态流转

```
待入库 → 已入库（统计入库）→ 已出库（出库）
```

- **待入库**：可编辑
- **已入库**：可编辑
- **已出库**：仅查看
